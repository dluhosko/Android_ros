This Javadoc documents the VibroTac Bluetooth API for Android.
The API provides an easy access via Bluetooth communication to your VibroTac (www.vibrotac.com) by SENSODRIVE.
This API supports
- VibroTac firmware since version 1.0.0. on
- VibroTac hardware since VibroTac VT04
- Android SDK since version 10
